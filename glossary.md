## Web Components
Web Components are a set of native web standards that allow developers to create reusable, encapsulated custom HTML elements using browser APIs rather than frameworks. They are composed of technologies like Custom Elements, Shadow DOM, and HTML Templates, which together enable components that manage their own structure, style, and behavior. Web Components are important because they are framework-agnostic, meaning the same component can be used in plain HTML, React, Vue, or Angular, making them ideal for design systems, shared UI libraries, and long-term maintainability.

https://developer.mozilla.org/en-US/docs/Web/Web_Components
https://webcomponents.org/introduction
https://developers.google.com/web/fundamentals/web-components

## Lit
Lit is a lightweight JavaScript library that simplifies building Web Components by adding reactivity, declarative templating, and efficient rendering. Built directly on web standards, Lit reduces the boilerplate required when working with Custom Elements and Shadow DOM while preserving their benefits. Lit is important for web development because it provides a framework-like developer experience without framework lock-in, making it a popular choice for scalable design systems and cross-framework component libraries.

https://lit.dev/docs/
https://lit.dev/why-lit/
https://opensource.google/projects/lit

## NPM
NPM (Node Package Manager) is the default package manager for Node.js and the largest ecosystem of open-source JavaScript libraries in the world. It allows developers to install, share, and manage dependencies needed for web applications, such as frameworks, build tools, and utilities. NPM is essential to modern web development because it enables modular development, speeds up workflows, and supports collaboration by standardizing how dependencies are distributed and versioned.

## JavaScript
JavaScript is a high-level, interpreted programming language that runs in the browser and on servers (via Node.js) to create interactive, dynamic web experiences. It enables functionality such as form validation, animations, API calls, and real-time updates, forming one of the three core web technologies alongside HTML and CSS. JavaScript is critical to web development because it powers client-side interactivity, modern frameworks, and full-stack applications.

## Git
Git is a distributed version control system that tracks changes in code over time, allowing developers to collaborate, experiment safely, and revert to previous versions when needed. It enables branching, merging, and collaborative workflows used by teams of all sizes. Git is important for web development because it provides code history, collaboration, and stability, and it underpins platforms like GitHub, GitLab, and Bitbucket.
